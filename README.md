---

## 📂 Organização do Projeto

O projeto está estruturado para facilitar a navegação por temas de aprendizado:

**Fundamentos (`Aula 02` e `Aula 03`)**: Operadores aritméticos, precedência, conversão de tipos e lógica sequencial aplicada (finanças, tempo e geometria).
**Estruturas de Repetição (`Aula While` e `main`)**: Domínio de laços `while` e `for`, acumuladores, contadores e validação de dados.
**Algoritmos Avançados (`Aula 14`)**: Implementação de lógica de números primos, fatoriais, juros compostos e simulações aleatórias.

## 💡 Destaques Técnicos

### 1. Tratamento de Exceções Matemáticas
Implementação de lógica para evitar erros de divisão por zero em cálculos de médias de números pares e ímpares, utilizando operadores ternários para maior fluidez do código.

### 2. Simulações e Negócios
**Projeção Salarial**: Comparação entre regimes de juros simples e compostos para progressão de carreira.
**Estatística Prática**: Algoritmos para análise de dados eleitorais e gestão de custos em agronegócio.
