## 📌 Sobre o Projeto
Este repositório centraliza meus estudos em **Engenharia de Software**, focando na maestria da linguagem Python. Aqui, transformo conceitos teóricos de algoritmos em soluções práticas, abrangendo desde fundamentos matemáticos até lógica complexa de repetição e automação de problemas reais.

---

## 📂 Organização do Projeto

O projeto está estruturado de forma evolutiva para facilitar a navegação pelos temas:

* **🌱 Fundamentos (`Aula 02` e `03`)**: Operadores aritméticos, precedência, conversão de tipos e lógica sequencial aplicada a finanças e geometria.
* **🔄 Estruturas de Repetição (`Aula While` e `main`)**: Domínio de laços `while` e `for`, uso de acumuladores, contadores e técnicas de validação de dados.
* **🧠 Algoritmos Avançados (`Aula 14`)**: Implementação de lógica de números primos, fatoriais, juros compostos e simulações aleatórias.

---

## 💡 Destaques Técnicos

### 1. Robustez e Tratamento de Dados
Implementação de lógica para evitar erros de divisão por zero em cálculos estatísticos e uso estratégico de **operadores ternários** para garantir um código mais limpo e fluido.

### 2. Simulações de Negócios e Estatística
* **Projeção Salarial**: Comparação algorítmica entre regimes de juros simples e compostos.
* **Gestão e Agronegócio**: Algoritmos voltados para análise de dados eleitorais e controle de custos operacionais.
